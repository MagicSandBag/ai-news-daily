## 🔴 待解决 - 线上问题

### 问题 1: pages-build-deployment 频繁执行

**现象**: GitHub Actions 显示 `pages-build-deployment` 自动执行多次

**原因分析**:
- 这是 GitHub Pages 的自动部署工作流
- 当 `docs/` 目录有变更时自动触发
- 每次推送新闻都会触发一次部署

**影响**: 消耗 GitHub Actions 额度，可能造成部署队列堆积

---

### 问题 2: 线上未调用 DeepSeek 翻译

**现象**: 线上版本的新闻没有中文翻译

**原因**: `DEEPSEEK_API_KEY` 未配置为 GitHub Secret

**修复方案**: ✅ 已修复
```yaml
# .github/workflows/daily-fetch.yml 已更新:
jobs:
  fetch-news:
    env:
      DEEPSEEK_API_KEY: ${{ secrets.DEEPSEEK_API_KEY }}
```

**操作步骤**:
1. GitHub 仓库 → Settings → Secrets and variables → Actions
2. New repository secret
3. Name: `DEEPSEEK_API_KEY`
4. Value: `sk-00ed249fa4ac4552a5a8729690fc6f9b`
5. 点击 "Add secret"

---
