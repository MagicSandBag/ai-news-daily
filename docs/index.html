Context Gateway 是一款由 YC 支持的 Compresr 公司开发的代理工具，旨在为 AI 智能体工作流提供即时历史压缩和上下文优化功能。该项目作为一个代理服务器，部署在用户的 AI 智能体（如 Claude Code、Cursor 等）与 LLM API 之间，核心目标是解决长对话场景下的上下文窗口限制问题。，其核心功能是背景式的即时历史压缩。当对话长度达到预设的触发阈值（默认为上下文窗口的 75%）时，系统会自动在后台对较早的对话历史进行压缩（例如生成摘要），而无需用户等待压缩过程完成，从而确保对话流畅不间断。用户可以通过检查日志文件 `history_compaction.jsonl` 来了解压缩的具体情况。，项目支持多种 AI 智能体，包括 Claude Code IDE 集成、Cursor IDE 集成、开源替代方案 OpenClaw，以及允许用户自定义配置的“custom”选项。快速启动过程简便，通过命令行安装 `gateway` 二进制文件后，运行 `context-gateway` 命令会启动一个交互式的 TUI 向导，引导用户完成配置：选择智能体类型、设置用于摘要的模型和 API 密钥、配置 Slack 通知（可选），以及调整触发压缩的阈值。，该工具的价值在于无缝优化 LLM 的上下文使用，通过预计算摘要来避免因触及上下文限制而导致的等待或中断，提升了 AI 编码助手等智能体在长周期、复杂任务中的连续性和效率。项目已在 GitHub 开源，采用 Apache-2.0 许可证，并设有 Discord 社区以欢迎贡献者参与。
