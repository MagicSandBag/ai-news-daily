人工智能在其他语言中的流利性掩盖了其西方世界观，可能误导用户。华盛顿大学人类学与亚洲研究教授加雷思·巴金指出，尽管ChatGPT、Claude和Gemini等大型语言模型（LLM）能以印尼语、阿拉伯语、斯瓦希里语或印地语等多种语言流利应答，但其底层推理逻辑仍根植于西方文化视角。这种“认识论持续性”意味着，即使AI语法完美、语调恰当，其输出的内容依然基于美国文化假设，例如强调个人自主、直接沟通，以及在家庭冲突中建议“划清界限”甚至断绝关系。，巴金的研究以印尼社会为例说明问题：当用户用印尼语咨询家庭纠纷处理时，AI给出的建议虽语言地道，却忽视了印尼社会更重视的共识构建、社会和谐与家庭集体动态等价值观。这种文化错位可能使许多用户难以察觉，从而接受不适用于本地情境的指导。，核心问题在于，当前主流AI模型的训练数据主要来自英语语料库，尤其是基于美国互联网内容。这导致模型在跨语言交互时，仅完成表层语言转换，而未能内化目标语言背后的文化语境与世界观。巴金将这种现象称为“流利不等于理解”——AI可以模仿语言形式，却无法真正把握文化内核。，该研究警示，随着AI在多语言场景中的普及，这种隐蔽的文化偏见可能强化西方中心主义，影响非英语用户在生活决策、文化认知乃至社会价值观层面的判断。解决这一挑战需推动训练数据的文化多样性，并开发能识别和适应本地价值观的AI系统。
