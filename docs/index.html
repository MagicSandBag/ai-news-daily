Anthropic公司近日向Claude Code订阅用户发出通知，宣布自太平洋时间4月4日中午12点起，将不再允许用户通过订阅额度使用包括OpenClaw在内的第三方集成工具。用户仍可使用其Claude账户登录这些工具，但相关使用将转为按量计费模式，费用独立于订阅计划之外。订阅服务将继续覆盖所有官方Claude产品，如Claude Code和Claude Cowork。，为缓解政策变动带来的影响，Anthropic将提供一笔一次性抵扣金，金额等同于用户的月度订阅费，需在4月17日前领取。同时，公司推出了预购额外使用量捆绑包的折扣方案，折扣最高可达30%。Anthropic解释称，此类第三方工具给其系统带来了超常负荷，公司需要谨慎管理容量资源，并优先保障核心产品用户的使用体验。受影响的用户也可选择在收到后续邮件后申请订阅退款。，社区讨论中指出，OpenClaw作为一种高度自动化的“强力用户”工具，其广泛使用可能对服务成本与系统稳定性构成压力。有观点认为，此次政策调整实质上是Anthropic为引导用户更多使用其官方平台工具、减少通过第三方工具消耗资源而采取的措施，而OpenClaw的高调存在为此提供了合适的调整理由。尽管存在使用限制（如5小时上限和每周限额），但部分用户指出，通过官方工具如Claude的循环功能同样可达到较高的令牌使用量，因此本次调整的重点并非单纯的技术容量管理，而涉及商业策略与平台生态控制。

研究发现“认知投降”导致AI用户放弃逻辑思考，宾夕法尼亚大学的研究人员在《思考——快速、缓慢与人工：AI如何重塑人类推理及认知投降的兴起》一文中提出，大型语言模型（LLM）工具的用户普遍存在“认知投降”现象，即用户将批判性思考外包给AI，不加审查地全盘接受其看似权威的答案。，研究基于认知双系统理论框架展开：系统1代表快速、直觉、情感驱动的处理；系统2代表缓慢、审慎、分析性的推理。研究人员认为，AI系统的出现催生了第三种决策类别——“人工认知”，即决策由源于算法系统而非人脑的外部、自动化、数据驱动的推理所主导。，与以往使用计算器或GPS等工具进行特定任务的“认知卸载”不同（人们会战略性地将部分工作委托给可靠算法，同时运用自身推理进行监督和评估），AI系统引发了一种性质不同的“认知投降”。在这种模式下，用户仅提供“最低限度的内部参与”，不加监督或验证地全盘接受AI的推理。研究指出，当LLM的输出“流畅、自信或阻力极小时”，这种“对推理本身不加批判的放弃”尤为常见。，为衡量这种认知投降的普遍性和影响，研究人员基于认知反射测试（CRT）进行了一系列实验。这类测试旨在诱使依赖“直觉”（系统1）的参与者给出错误答案，但对运用更“审慎”（系统2）思考过程者则相对简单。实验发现，咨询AI的测试对象绝大多数都愿意不加审查地接受其答案，无论答案正确与否。时间压力和外部激励等因素会进一步影响用户依赖AI的倾向。，研究强调，AI输出的流畅性和自信表象可能掩盖其潜在的事实或逻辑错误，导致用户过度依赖并放弃自身的逻辑判断。这凸显了在AI工具日益普及的背景下，培养用户保持批判性思维和进行人工监督的重要性。
