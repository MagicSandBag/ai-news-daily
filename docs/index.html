Claude Code 近期更新了其交互式开发环境的速查指南，新增多项核心功能与优化。主要更新包括：新增 `--bare` 标志以启用极简无头模式（无钩子/LSP/插件），引入 `--channels` 用于权限中继和 MCP 推送消息（预览功能），为技能和斜杠命令添加 effort 前置元数据，将 `/fork` 重命名为 `/branch`（保留别名），以及 SendMessage 功能可自动恢复已停止的代理。，键盘快捷键方面，提供了全面的会话控制组合：Ctrl+C 取消输入/生成，Ctrl+D 退出会话，Ctrl+L 清屏，Ctrl+O 切换详细输出，Ctrl+R 反向搜索历史记录，Ctrl+G 在编辑器中打开提示，Ctrl+B 后台运行任务，Ctrl+T 切换任务列表，Ctrl+V 粘贴图像，双击 Ctrl+F 终止后台代理，Esc+Esc 回退/撤销。模式切换支持 Shift+Tab 循环权限模式，Alt+P 切换模型，Alt+T 切换思考模式。输入时使用 `\`+Enter 快速换行，Ctrl+J 控制序列换行。前缀命令包括 `/` 执行斜杠命令，`!` 直接运行 bash，`@` 提及文件并自动补全。会话选择器可通过方向键导航、展开/折叠、预览、重命名或搜索，支持查看所有项目或当前分支。，MCP 服务器管理新增多种传输方式：`--transport http`（推荐远程 HTTP）、`--transport stdio`（本地进程）、`--transport sse`（远程 SSE）。配置作用域涵盖本地项目（.claude.json）、项目共享（.mcp.json）和用户全局（~/.claude.json）。管理命令包括 `/mcp` 交互式界面、`claude mcp list` 列出所有服务器、`claude mcp serve` 将 Claude Code 作为 MCP 服务器运行，以及新增的“启发式服务器”可在任务中途请求输入。，斜杠命令体系大幅扩展，涵盖会话管理（如 `/clear` 清空对话、`/compact` 压缩上下文、`/resume` 恢复会话、`/branch` 分支对话）、配置操作（如 `/config` 打开设置、`/model` 切换模型、`/effort` 设置工作强度）、工具集成（如 `/mcp` 管理服务器、`/hooks` 管理钩子、`/agents` 管理代理）以及特殊功能（如 `/plan` 进入规划模式、`/voice` 支持 20 种语言的即按即说语音输入、`/pr-comments` 获取 GitHub PR 评论）。新增命令 `/remote-control` 可将终端桥接至 claude.ai/code，`/security-review` 提供变更安全分析。，记忆与文件系统通过 CLAUDE.md 实现，支持项目级（./CLAUDE.md，团队共享）和用户全局级（~/.claude/CLAUDE.md）配置，增强了上下文管理和协作能力。
