独立前端开发者Ankur Sethi在一月到二月间的四周内，利用Claude Code构建了一门全新的编程语言，并将其命名为Cutlet（以其宠物猫命名）。该项目源代码已公开在GitHub上，包含构建说明和示例程序。作者自2021年GitHub Copilot发布以来一直使用LLM辅助编程，但此前仅将其用于生成样板代码或进行针对性修改。而在Cutlet的开发过程中，作者尝试完全由Claude生成每一行代码，自身并未阅读任何生成内容，而是通过设置防护机制确保代码正确运行。，实验成果令人惊讶：Cutlet已成功实现，可在macOS和Linux系统上构建并运行，能够执行真实程序。尽管其内部可能隐藏错误，但作者认为这与任何仅开发四周的编程语言可能存在的问题相当。，Cutlet具备动态语言特性：支持数组和字符串操作，变量通过my关键字声明，变量名可包含连字符（语法规则类似Raku），目前数值类型仅支持双精度浮点数。其特色功能包括@元运算符——可将任意常规二元运算符转换为对数组的向量化操作，例如对数组每个元素进行乘法和加法运算。@:运算符则用于将两个数组压缩为映射结构。此外，内置say函数用于输出文本（返回nothing值，即Cutlet的null表示）。，更巧妙的设计体现在：@元运算符支持比较操作生成布尔数组，而该布尔数组可直接作为索引对原数组进行过滤筛选，例如快速提取符合温度条件的城市名称。这些特性展示了Cutlet在数据操作方面的简洁表达能力。，该项目不仅验证了LLM在完整编程语言构建中的潜力，也引发了开发者对AI时代编程职业形态的思考。通过将开发过程转化为对LLM输出的约束与验证，作者探索了一种新型的人机协作编程范式。
