Understudy是一款可教学的桌面智能体，其核心理念是让用户通过单次演示来教会AI助手完成日常电脑操作任务。该项目旨在解决当前AI工具覆盖工作场景有限的问题，因为用户的日常工作往往分散在浏览器、桌面应用、终端和通讯工具等多种界面割裂的软件中。，Understudy的设计模仿人类同事的学习与成长过程，其架构分为五个渐进层次：第一层是原生操作软件，智能体可以像人一样查看、点击、输入和验证任何应用程序；第二层是从演示中学习，用户演示一次任务，智能体便能提取操作意图（而非仅仅记录坐标），记住成功路径；第三层是记忆与独立执行，智能体能记住常规流程并自主完成；第四层是发现优化路径，经过一段时间运行后，它能自主发现更快的执行方式；第五层是主动预测与执行，智能体最终能够预判需求并主动采取行动。这五个阶段对应着一个“新同事”从入职观察（Day 1）、模仿询问（Week 1）、独立执行（Month 1）、寻找捷径（Month 3）到主动预判（Month 6）的成长历程，这也是其命名为“Understudy”（替角）的寓意——如同戏剧中的替补演员，通过观察主角来学习角色，并在需要时接手工作。，该项目的关键优势在于无需为每个软件进行专门的API集成，也无需使用复杂的工作流构建器。它在一个本地运行时环境中统一操作图形用户界面（GUI）、浏览器、命令行和文件系统。用户只需演示一次，智能体便能理解任务意图，记住成功路径，并随着时间推移发现更优的执行路线，最终自主处理常规工作。
