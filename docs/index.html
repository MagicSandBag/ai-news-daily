近期《福布斯》关于Cursor的文章中声称，Anthropic每月200美元的Claude Code Max计划可能消耗高达5000美元的计算资源，这一说法在社交媒体上广泛传播。然而，这一数字经不起基本推敲。实际上，该数据很可能混淆了零售API价格与实际计算成本。，Anthropic目前对Opus 4.6的API定价为每百万输入token 5美元、每百万输出token 25美元。按照这一价格，重度Claude Code Max用户确实可能产生相当于5000美元的API使用量。但API定价并不等同于Anthropic提供这些token的实际成本。，通过OpenRouter平台上的开放权重模型定价可以更准确地估算推理成本。以架构规模与Opus 4.6相近的Qwen 3.5 397B-A17B模型为例，其在OpenRouter（通过阿里云提供）的定价为每百万输入token 0.39美元、输出token 2.34美元。相比之下，Opus 4.6的API价格高出约10倍。另一个可比模型Kimi K2.5的定价更为低廉，为每百万输入token 0.45美元、输出token 2.25美元。缓存token的定价差异同样显著：DeepInfra对Kimi K2.5的缓存读取收费为每百万token 0.07美元，而Anthropic的对应价格为0.50美元。，OpenRouter上的供应商需要覆盖计算成本、GPU费用并保持利润空间，他们并非慈善机构。如果多家供应商都能以Anthropic API价格约10%的水平提供规模相当的模型并持续运营，那么认为他们全部承受巨额亏损的说法难以成立。，因此，若一名重度Claude Code Max用户按照Anthropic零售API价格消耗了价值5000美元的token，实际计算成本可能仅为其10%左右，即约500美元。这意味着即使对极端重度用户，Anthropic每月可能损失约300美元，而非4800美元。此外，绝大多数用户远未达到使用上限。Anthropic在推出每周使用上限时曾表示，受影响用户比例不足5%。实际使用情况显示，多数用户仅消耗其token预算的一部分，远未达到理论极限值。
